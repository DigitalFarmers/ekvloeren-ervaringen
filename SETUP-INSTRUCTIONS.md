### Step 2: Upload to DirectAdmin

2. Navigate to `public_html` folder
3. Upload everything from the `out` folder
4. Upload the `.htaccess` file to `public_html`

### Step 3: Setup Database
1. In DirectAdmin, go to **MySQL Management**
2. Create new database: `ekvloeren_db`
3. Create new user: `ekvloeren_user`
4. Import the SQL files from `scripts/` folder:
   - `001-create-tables.sql`
   - `002-update-schema.sql`

### Step 4: Configure Environment
Create a `.env` file in your hosting root:
```env
DATABASE_URL=mysql://ekvloeren_user:password@localhost/ekvloeren_db
ADMIN_PASSWORD=your_secure_password
```

### Step 5: Test Your Site
Visit your domain and test:
- Homepage loads correctly
- Contact form works
- Admin dashboard accessible at `/admin`
